## 🚀 Sobre o Projeto

O **Albry Studio** não é apenas um portfólio; é uma máquina de aquisição de clientes. Construído para resolver o problema de marcas que possuem um excelente serviço, mas falham em comunicar seu valor através do digital.

O projeto foi meticulosamente desenhado para:
- Passar máxima **autoridade e sofisticação** através de um design premium e responsivo.
- Oferecer **velocidade extrema** e SEO perfeito (comprovado por métricas 100/100 no Lighthouse).
- Centralizar orçamentos de Identidade Visual e Sites de Alta Performance em um funil altamente otimizado.

---

## 🏗️ Arquitetura e Features

Este projeto utiliza o que há de mais moderno no ecossistema front-end para garantir escalabilidade, performance e manutenibilidade empresarial:

- ⚡ **Next.js (App Router):** Renderização no lado do servidor (SSR) e geração estática para máximo desempenho e SEO.
- 🎨 **Tailwind CSS:** Estilização utilitária e design system escalável, focando em customização rápida e consistente.
- 🎬 **Framer Motion:** Animações fluidas e otimizadas que não comprometem o First Contentful Paint (FCP) ou a renderização inicial.
- 🔒 **TypeScript & Zod:** Tipagem estática forte e validação de schemas ponta a ponta (end-to-end type safety).
- 📈 **Performance & Analytics:** Integração nativa com `@next/third-parties/google` e otimização cirúrgica de Core Web Vitals (LCP, FCP e TBT).

## 💻 Como Rodar Localmente

Para clonar e rodar esta aplicação em sua máquina, você precisará do [Git](https://git-scm.com) e do [Node.js](https://nodejs.org/en/download/) instalados.

Na sua linha de comando:

```bash
# Clone este repositório
$ git clone https://github.com/Bryanninja/alex-bryan-portfolio.git

# Entre no repositório
$ cd alex-bryan-portfolio

# Instale as dependências
$ npm install

# Rode a aplicação em modo de desenvolvimento
$ npm run dev
```

> **Aviso:** A aplicação abrirá automaticamente no endereço `http://localhost:3000`.
